# Rack Structure Rules - 01 Fundamentals

## Struktur Minimum Wajib

```text
01_fundamentals/
|-- README.md
|-- docs/
|   |-- README.md
|   |-- rack-scope.md
|   |-- rack-structure-rules.md
|   |-- rack-writing-guidelines.md
|   `-- rack-contribution-guidelines.md
`-- book-xx-.../
```

## Aturan Penamaan Buku

- gunakan format: `book-NN-topic-name`
- `NN` harus dua digit (`01`, `02`, ...)
- `topic-name` gunakan lowercase dan `-`
