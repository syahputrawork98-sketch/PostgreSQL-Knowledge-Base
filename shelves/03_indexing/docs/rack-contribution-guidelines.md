# Rack Contribution Guidelines - 03 Indexing

## Aturan Kontribusi Level Rak

1. Perubahan daftar buku harus memperbarui `README.md` rak.
2. Penambahan buku baru harus mengikuti format penamaan rak.
3. Perubahan aturan rak harus dicatat pada dokumen `docs/` rak terkait.
4. Hindari mengubah aturan internal buku dari level rak.

## Checklist PR Rak

- [ ] Struktur rak tetap valid
- [ ] Katalog buku sudah diperbarui
- [ ] Urutan baca tetap konsisten
- [ ] Tidak melanggar scope root atau buku
