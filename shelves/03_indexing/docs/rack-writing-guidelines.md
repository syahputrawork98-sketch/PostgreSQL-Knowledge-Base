# Rack Writing Guidelines - 03 Indexing

## Tujuan README Rak

README rak menjawab 3 hal:
1. apa yang dipelajari pada rak ini
2. buku apa saja yang tersedia
3. urutan baca yang direkomendasikan

## Format Minimum README Rak

- Ringkasan topik rak
- Daftar buku + deskripsi singkat
- Urutan baca
- Catatan prasyarat (jika ada)

## Gaya Penulisan

- ringkas dan langsung ke poin
- konsisten istilah teknis PostgreSQL
- hindari detail teknis tingkat buku di README rak
