# Ringkasan Tipe Data Dasar

Tipe data yang paling sering dipakai pemula:

| Tipe Data | Cocok Untuk | Contoh |
| --------- | ----------- | ------ |
| `INTEGER` | angka bulat | jumlah item |
| `NUMERIC` | angka desimal presisi | harga |
| `TEXT` | teks bebas | nama |
| `BOOLEAN` | status benar/salah | aktif |
| `DATE` | tanggal | tanggal lahir |
| `TIMESTAMP` | tanggal dan waktu | waktu dibuat |
