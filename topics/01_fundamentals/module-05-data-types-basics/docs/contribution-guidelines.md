# Contribution Guidelines (Module)

Aturan kontribusi untuk module ini:

1. Perubahan materi wajib memperbarui `CHANGELOG.md`.
2. Contoh SQL baru dapat disimpan di `examples/`.
3. Perubahan harus tetap pada scope tipe data dasar untuk pemula.
4. Bahasan yang terlalu dalam tentang tipe data lanjutan sebaiknya dipindahkan ke topic atau module lain yang lebih tepat.
