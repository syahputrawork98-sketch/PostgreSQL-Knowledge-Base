# Writing Guidelines (Module)

Pedoman penulisan module ini:

1. Jelaskan tipe data berdasarkan arti nilai yang disimpan.
2. Utamakan tipe data yang paling sering dipakai pemula.
3. Gunakan contoh SQL pendek dan mudah dibaca.
4. Sertakan tabel visual atau ringkasan cepat untuk memudahkan perbandingan.
5. Hindari detail teknis yang terlalu dalam pada tahap fundamentals.
6. Tambahkan kesalahan umum pemula saat memilih tipe data.
