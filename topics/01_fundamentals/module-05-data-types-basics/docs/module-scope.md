# Module Scope

Dokumen ini mengatur batas aturan pada module ini saja.

Cakupan:
- pengenalan tipe data dasar PostgreSQL
- perbedaan tipe data inti untuk pemula
- contoh pemilihan tipe data pada column
- kesalahan umum dalam memilih tipe data

Tidak mencakup:
- tipe data lanjutan secara mendalam
- optimisasi storage detail
- domain, enum, dan custom type tingkat lanjut
- detail timezone dan precision tingkat lanjut
