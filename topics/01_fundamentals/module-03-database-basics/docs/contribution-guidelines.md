# Contribution Guidelines (Module)

Aturan kontribusi untuk module ini:

1. Perubahan materi wajib memperbarui `CHANGELOG.md`.
2. Contoh SQL atau command baru dapat disimpan di `examples/`.
3. Perubahan harus tetap pada scope pengenalan database dan operasi dasarnya.
4. Hindari pembahasan detail yang lebih cocok untuk topic administration atau security.
