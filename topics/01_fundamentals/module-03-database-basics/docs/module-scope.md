# Module Scope

Dokumen ini mengatur batas aturan pada module ini saja.

Cakupan:
- definisi database dalam PostgreSQL
- posisi database dalam struktur server PostgreSQL
- perbedaan database, schema, dan table
- lifecycle dasar database
- operasi dasar yang relevan untuk pemula

Tidak mencakup:
- administrasi database tingkat lanjut
- backup dan restore detail
- permission management mendalam
- optimisasi dan tuning level database
