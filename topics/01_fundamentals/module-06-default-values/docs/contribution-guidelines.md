# Contribution Guidelines (Module)

Aturan kontribusi untuk module ini:

1. Perubahan materi wajib memperbarui `CHANGELOG.md`.
2. Contoh SQL baru dapat disimpan di `examples/`.
3. Perubahan harus tetap pada scope `DEFAULT` dan nilai bawaan dasar.
4. Bahasan yang dominan tentang constraint lanjutan atau otomasi lanjutan harus dipindahkan ke module yang lebih tepat.
