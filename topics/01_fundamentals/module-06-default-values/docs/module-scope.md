# Module Scope

Dokumen ini mengatur batas aturan pada module ini saja.

Cakupan:
- pengenalan `DEFAULT` pada column PostgreSQL
- contoh nilai bawaan sederhana
- pengaruh `DEFAULT` saat `INSERT`
- kesalahan umum saat memakai nilai bawaan

Tidak mencakup:
- constraint secara mendalam
- generated columns
- trigger atau otomatisasi lanjutan
- desain data tingkat lanjut
