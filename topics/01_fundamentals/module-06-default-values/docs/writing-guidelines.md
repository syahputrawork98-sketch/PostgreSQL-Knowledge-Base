# Writing Guidelines (Module)

Pedoman penulisan module ini:

1. Jelaskan `DEFAULT` dengan bahasa sederhana dan praktis.
2. Gunakan contoh yang mudah diuji langsung oleh pemula.
3. Bedakan dengan jelas `DEFAULT` dari constraint lain.
4. Analogi ringan boleh dipakai bila membantu menjelaskan nilai bawaan.
5. Hindari pembahasan fitur otomatisasi yang lebih lanjut.
6. Tambahkan kesalahan umum pemula saat memakai nilai bawaan.
