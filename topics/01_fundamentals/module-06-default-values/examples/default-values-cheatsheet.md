# Ringkasan Default Values

Contoh default yang sering dipakai:

| Pola | Arti |
| ---- | ---- |
| `BOOLEAN DEFAULT false` | status awal belum aktif atau belum selesai |
| `BOOLEAN DEFAULT true` | status awal aktif |
| `INTEGER DEFAULT 0` | nilai awal nol |
| `DATE DEFAULT CURRENT_DATE` | tanggal otomatis hari ini |
| `TIMESTAMP DEFAULT CURRENT_TIMESTAMP` | waktu otomatis saat data dibuat |
