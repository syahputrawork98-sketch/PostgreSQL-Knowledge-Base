# Contribution Guidelines (Module)

Aturan kontribusi untuk module ini:

1. Perubahan materi wajib memperbarui `CHANGELOG.md`.
2. Contoh file baru dapat disimpan di `examples/`.
3. Perubahan harus tetap pada scope seed, import, dan export dasar.
4. Bahasan yang dominan tentang backup, restore, atau pipeline data lanjutan harus dipindahkan ke topic lain yang lebih tepat.
