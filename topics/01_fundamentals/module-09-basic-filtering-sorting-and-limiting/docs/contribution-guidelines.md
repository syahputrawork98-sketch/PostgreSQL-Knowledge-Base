# Contribution Guidelines (Module)

Aturan kontribusi untuk module ini:

1. Perubahan materi wajib memperbarui `CHANGELOG.md`.
2. Contoh SQL baru dapat disimpan di `examples/`.
3. Perubahan harus tetap pada scope filtering, sorting, dan limiting dasar.
4. Bahasan yang dominan tentang join, agregasi, atau query lanjutan harus dipindahkan ke topic yang lebih tepat.
