# Writing Guidelines (Module)

Pedoman penulisan module ini:

1. Gunakan query yang pendek dan mudah diikuti.
2. Jelaskan perubahan tujuan query saat `WHERE`, `ORDER BY`, `LIMIT`, atau `OFFSET` ditambahkan.
3. Hindari masuk ke query lanjutan di luar scope fundamentals.
4. Gunakan contoh yang menunjukkan hasil query menjadi lebih terarah.
5. Tambahkan kesalahan umum pemula saat mulai memfilter dan mengurutkan data.
