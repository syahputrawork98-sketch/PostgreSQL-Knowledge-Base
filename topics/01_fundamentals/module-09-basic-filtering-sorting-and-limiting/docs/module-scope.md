# Module Scope

Dokumen ini mengatur batas aturan pada module ini saja.

Cakupan:
- penggunaan dasar `WHERE`
- penggunaan dasar `ORDER BY`
- penggunaan dasar `LIMIT`
- pengenalan dasar `OFFSET`
- kombinasi query sederhana

Tidak mencakup:
- join
- group by dan agregasi mendalam
- subquery
- kondisi query yang kompleks
