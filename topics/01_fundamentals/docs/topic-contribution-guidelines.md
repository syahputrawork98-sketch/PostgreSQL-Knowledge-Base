# Topic Contribution Guidelines - 01 Fundamentals

1. Perubahan daftar module wajib memperbarui `README.md` topic.
2. Penambahan module baru wajib mengikuti aturan penamaan.
3. Perubahan aturan topic wajib didokumentasikan di `docs/` topic.
4. Jangan mengubah aturan internal module dari level topic.
