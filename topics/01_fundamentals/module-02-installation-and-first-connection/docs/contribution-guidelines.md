# Contribution Guidelines (Module)

Aturan kontribusi untuk module ini:

1. Perubahan materi wajib memperbarui `CHANGELOG.md`.
2. Contoh command atau SQL baru dapat disimpan di `examples/`.
3. Perubahan harus tetap pada scope setup awal dan koneksi pertama.
4. Hindari membuat instruksi yang terlalu spesifik ke satu platform jika tidak benar-benar perlu.
