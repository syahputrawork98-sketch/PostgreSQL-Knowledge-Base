# Contribution Guidelines (Module)

Aturan kontribusi untuk module ini:

1. Perubahan materi wajib memperbarui `CHANGELOG.md`.
2. Contoh SQL baru dapat disimpan di `examples/`.
3. Perubahan harus tetap pada scope CRUD dasar untuk pemula.
4. Bahasan yang dominan tentang filtering lanjutan, join, atau transaksi harus dipindahkan ke module atau topic lain yang lebih tepat.
