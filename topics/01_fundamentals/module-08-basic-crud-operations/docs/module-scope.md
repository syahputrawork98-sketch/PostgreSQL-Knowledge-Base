# Module Scope

Dokumen ini mengatur batas aturan pada module ini saja.

Cakupan:
- pengenalan CRUD dasar
- contoh `INSERT`, `SELECT`, `UPDATE`, dan `DELETE`
- penekanan pada penggunaan `WHERE`
- kebiasaan aman saat memanipulasi data

Tidak mencakup:
- join lanjutan
- filtering dan sorting detail
- transaksi secara mendalam
- bulk operations lanjutan
