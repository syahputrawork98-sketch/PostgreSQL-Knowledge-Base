# Checklist Setelah Menyelesaikan Fundamentals

- Apakah saya sudah paham database, schema, table, row, dan column?
- Apakah saya sudah bisa membuat table sederhana sendiri?
- Apakah saya sudah cukup nyaman dengan `INSERT`, `SELECT`, `UPDATE`, dan `DELETE`?
- Apakah saya sudah mencoba `WHERE`, `ORDER BY`, dan `LIMIT`?
- Apakah saya punya database latihan untuk lanjut ke topic query?
- Jika belum yakin di salah satu bagian, module mana yang perlu saya ulang?
