# Contribution Guidelines (Module)

Aturan kontribusi untuk module ini:

1. Perubahan materi wajib memperbarui `CHANGELOG.md`.
2. Aset roadmap atau ringkasan baru dapat disimpan di `assets/` atau `examples/` jika diperlukan.
3. Perubahan harus tetap pada scope transisi dari fundamentals ke topic berikutnya.
4. Hindari mengubah module ini menjadi pembahasan teknis detail dari topic lanjutan.
