# Module Scope

Dokumen ini mengatur batas aturan pada module ini saja.

- rangkuman capaian fundamentals
- indikator kesiapan untuk lanjut

- pengantar ke topic selanjutnya

Tidak mencakup:
- pembahasan detail topic lanjutan
- silabus lengkap untuk semua topic berikutnya
- strategi karier atau sertifikasi
