# Contribution Guidelines (Module)

Aturan kontribusi untuk module ini:

1. Perubahan materi wajib memperbarui `CHANGELOG.md`.
2. Contoh SQL baru dapat disimpan di `examples/`.
3. Perubahan harus tetap pada scope schema, table, row, dan column.
4. Bahasan yang dominan tentang tipe data, constraint, atau query harus dipindahkan ke module yang lebih tepat.
