# Checklist Singkat Sebelum Menjalankan Query

- Apakah saya sedang berada di database yang benar?
- Apakah saya paham table yang sedang saya ubah?
- Apakah tipe data column sudah sesuai?
- Apakah query `UPDATE` atau `DELETE` saya memakai `WHERE`?
- Apakah saya akan mengecek hasil dengan `SELECT` setelahnya?
- Apakah saya sedang membuat query terlalu rumit untuk tahap ini?
