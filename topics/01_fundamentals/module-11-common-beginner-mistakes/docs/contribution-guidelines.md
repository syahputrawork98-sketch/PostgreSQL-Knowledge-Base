# Contribution Guidelines (Module)

Aturan kontribusi untuk module ini:

1. Perubahan materi wajib memperbarui `CHANGELOG.md`.
2. Contoh ringkasan atau checklist baru dapat disimpan di `examples/`.
3. Perubahan harus tetap pada scope kesalahan umum pemula dan kebiasaan aman.
4. Hindari memperluas module ini menjadi troubleshooting teknis tingkat lanjut.
