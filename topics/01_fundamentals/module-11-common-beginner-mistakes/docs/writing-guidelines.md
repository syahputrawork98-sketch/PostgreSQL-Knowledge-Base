# Writing Guidelines (Module)

Pedoman penulisan module ini:

1. Fokus pada kesalahan yang benar-benar sering terjadi pada pemula.
2. Jelaskan kenapa kesalahan itu muncul dan bagaimana menghindarinya.
3. Gunakan bahasa yang menenangkan, bukan menghakimi.
4. Visual ringan boleh dipakai untuk merangkum kesalahan utama.
5. Jadikan module ini sebagai pengikat dari materi fundamentals sebelumnya.
