# Writing Guidelines (Module)

Pedoman penulisan module ini:

1. Jelaskan konsep dasar dengan bahasa sederhana.
2. Sertakan contoh SQL pendek dan bertahap.
3. Gunakan istilah PostgreSQL secara konsisten.
4. Tambahkan catatan kesalahan umum pemula.
5. Gunakan analogi singkat untuk membantu konsep abstrak.
6. Sertakan minimal satu SVG atau diagram ringan yang memperjelas hubungan object dasar.
