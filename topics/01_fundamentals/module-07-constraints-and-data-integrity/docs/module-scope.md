# Module Scope

Dokumen ini mengatur batas aturan pada module ini saja.

Cakupan:
- pengenalan constraint dasar PostgreSQL
- hubungan constraint dengan integritas data
- contoh `NOT NULL`, `UNIQUE`, `PRIMARY KEY`, `FOREIGN KEY`, dan `CHECK`
- kesalahan umum saat memakai constraint

Tidak mencakup:
- constraint naming convention mendalam
- desain relasi tingkat lanjut
- cascade behavior secara mendalam
- normalisasi dan desain skema lanjutan
