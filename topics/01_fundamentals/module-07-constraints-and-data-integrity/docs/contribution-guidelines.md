# Contribution Guidelines (Module)

Aturan kontribusi untuk module ini:

1. Perubahan materi wajib memperbarui `CHANGELOG.md`.
2. Contoh SQL baru dapat disimpan di `examples/`.
3. Perubahan harus tetap pada scope constraint dasar dan integritas data.
4. Bahasan yang terlalu dalam tentang desain relasi lanjutan atau administrasi schema harus dipindahkan ke module atau topic lain yang lebih tepat.
