# Ringkasan Constraint Dasar

| Constraint | Fungsi Singkat |
| ---------- | -------------- |
| `NOT NULL` | nilai tidak boleh kosong |
| `UNIQUE` | nilai tidak boleh duplikat |
| `PRIMARY KEY` | identitas utama row |
| `FOREIGN KEY` | menjaga relasi antar table |
| `CHECK` | memastikan kondisi logis terpenuhi |
