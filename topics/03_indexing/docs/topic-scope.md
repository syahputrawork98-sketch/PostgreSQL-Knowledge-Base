# Topic Scope - 03 Indexing

Dokumen ini menetapkan batas aturan untuk topic `03_indexing`.

## Berlaku Untuk

- `topics/03_indexing/README.md`
- `topics/03_indexing/docs/*`
- daftar module di dalam `topics/03_indexing/`

## Tidak Berlaku Untuk

- aturan global repositori (lihat `docs/` root)
- aturan isi internal tiap module (lihat `docs/` module)

## Prinsip

1. Topic mengatur organisasi topik indexing.
2. Topic tidak memaksakan satu template isi ke semua module.
3. Topic menentukan urutan belajar antar module secara logis.
