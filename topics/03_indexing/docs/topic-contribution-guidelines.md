# Topic Contribution Guidelines - 03 Indexing

## Aturan Kontribusi Level Topic

1. Perubahan daftar module harus memperbarui `README.md` topic.
2. Penambahan module baru harus mengikuti format penamaan topic.
3. Perubahan aturan topic harus dicatat pada dokumen `docs/` topic terkait.
4. Hindari mengubah aturan internal module dari level topic.

## Checklist PR Topic

- [ ] Struktur topic tetap valid
- [ ] Katalog module sudah diperbarui
- [ ] Urutan baca tetap konsisten
- [ ] Tidak melanggar scope root atau module
