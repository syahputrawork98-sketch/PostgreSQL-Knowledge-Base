# Topic Structure Rules - 03 Indexing

## Struktur Minimum Wajib

```text
03_indexing/
|-- README.md
|-- docs/
|   |-- README.md
|   |-- topic-scope.md
|   |-- topic-structure-rules.md
|   |-- topic-writing-guidelines.md
|   `-- topic-contribution-guidelines.md
`-- module-xx-.../
```

## Aturan Penamaan Module

- gunakan format: `module-NN-topic-name`
- `NN` harus dua digit (`01`, `02`, ...)
- `topic-name` gunakan lowercase dan `-`

## Aturan Katalog Topic

1. Semua module yang aktif harus tercantum di `README.md` topic.
2. Urutan module mengikuti progres pembelajaran dari dasar ke lanjutan.
3. Module yang belum siap dipublikasikan tidak masuk katalog utama.
