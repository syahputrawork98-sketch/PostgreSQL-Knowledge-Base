# Topic Writing Guidelines - 03 Indexing

## Tujuan README Topic

README topic menjawab 3 hal:
1. apa yang dipelajari pada topic ini
2. module apa saja yang tersedia
3. urutan baca yang direkomendasikan

## Format Minimum README Topic

- Ringkasan topik
- Daftar module + deskripsi singkat
- Urutan baca
- Catatan prasyarat (jika ada)

## Gaya Penulisan

- ringkas dan langsung ke poin
- konsisten istilah teknis PostgreSQL
- hindari detail teknis tingkat module di README topic
