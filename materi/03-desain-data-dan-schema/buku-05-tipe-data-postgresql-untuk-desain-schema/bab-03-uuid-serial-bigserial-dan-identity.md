## 1. Ringkasan Bab
Membahas desain Primary Key tergenerasi otomatis menggunakan SERIAL/BIGSERIAL, standar SQL baru IDENTITY, serta GUID/UUID untuk sistem terdistribusi.

---

## 2. Tujuan Belajar
Setelah menyelesaikan bab ini, pembaca diharapkan mampu:
- Memahami perbedaan mekanisme SERIAL (sequences) vs GENERATED AS IDENTITY (SQL Standard).
- Menguasai perbandingan keuntungan dan kelemahan BIGINT Auto-increment vs UUID sebagai Primary Key.
