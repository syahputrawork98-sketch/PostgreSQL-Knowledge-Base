# Jalur Belajar: Level 0 - 4 (Dasar ke Menengah)

Status: SKELETON

Tujuan Jalur Belajar:
Membantu pembaca dari nol hingga mampu merancang database dan mengintegrasikannya dengan aplikasi backend (Backend Developer Ready).

## Tahapan Belajar:

### 1. Tahap Orientasi (Level 0 - 1)
- **Rak**: [01 - Orientasi, Sejarah, dan Fondasi PostgreSQL](../01-orientasi-sejarah-dan-fondasi-postgresql/)
- **Fokus**: Memahami apa itu PostgreSQL, sejarahnya, dan mental model database relasional.

### 2. Tahap Querying Dasar (Level 1 - 2)
- **Rak**: [02 - SQL dan Querying](../02-sql-dan-querying/)
- **Fokus**: Mahir menulis perintah SELECT, filtering, sorting, dan penggabungan data (JOIN).

### 3. Tahap Desain Data (Level 2 - 3)
- **Rak**: [03 - Desain Data dan Schema](../03-desain-data-dan-schema/)
- **Fokus**: Mampu membuat tabel, menentukan tipe data, menjaga integritas dengan constraint, dan memahami normalisasi.

### 4. Tahap Implementasi Aplikasi (Level 3 - 4)
- **Rak**: [04 - PostgreSQL untuk Aplikasi](../04-postgresql-untuk-aplikasi/)
- **Fokus**: Menghubungkan aplikasi ke database, mengelola migrasi skema, dan menulis query untuk fitur aplikasi nyata.
