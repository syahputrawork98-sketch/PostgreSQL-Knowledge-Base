# Buku 01: Orientasi PostgreSQL

*   **Status Buku**: **DRAFT / REVIEW**
*   **Rak**: [Rak 01: Orientasi, Sejarah, dan Fondasi PostgreSQL](../README.md)
*   **Target Level**: Level 0 - Level 1 (Dasar)
*   **Tipe Materi**: Teoretis & Konseptual

---

## 1. Tujuan Buku
Buku ini bertujuan untuk mengenalkan PostgreSQL dari sisi fungsionalitas utama, memahami mengapa PostgreSQL disebut sebagai Object-Relational Database Management System (ORDBMS), serta memetakan keunggulan dan fiturnya di antara kompetitor database industri modern lainnya.

---

## 2. Hasil Belajar
Setelah menyelesaikan buku ini, pembaca diharapkan mampu menjelaskan karakteristik PostgreSQL, membandingkannya dengan engine RDBMS lain (MySQL, Oracle, dll.), serta menyadari perannya dalam ekosistem sistem backend modern.

---

## 3. Kebijakan Struktur (No Fixed Limit)
Jumlah bab dalam buku ini dikembangkan mengikuti kedalaman pemahaman orientasi dasar dan tidak dibatasi secara kaku. Buku ini dapat berkembang secara berkala untuk mencakup variasi komparasi produk database baru.
