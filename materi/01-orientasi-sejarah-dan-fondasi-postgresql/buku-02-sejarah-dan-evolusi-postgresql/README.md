# Buku 02: Sejarah dan Evolusi PostgreSQL

*   **Status Buku**: **SKELETON** (Belum Ditulis)
*   **Rak**: [Rak 01: Orientasi, Sejarah, dan Fondasi PostgreSQL](../README.md)
*   **Target Level**: Level 0 - Level 1 (Dasar)
*   **Tipe Materi**: Sejarah & Perkembangan Engine

---

## 1. Tujuan Buku
Buku ini dirancang untuk mendokumentasikan sejarah perkembangan mesin PostgreSQL dari proyek Ingres yang diinisiasi oleh Michael Stonebraker, perubahan nama menjadi Postgres, penambahan interpreter SQL (Postgres95), hingga evolusi fitur utama pada versi-versi rilis penting (PostgreSQL v1.0 s/d v17).

---

## 2. Hasil Belajar (Direncanakan)
Setelah menyelesaikan buku ini, pembaca diharapkan mampu menjelaskan akar historis dari keputusan arsitektur mesin PostgreSQL dan bagaimana standar fitur baru diadopsi seiring berjalannya waktu.

---
