# Contribution Guidelines (Global)

Dokumen ini mengatur kontribusi level repositori.

## Aturan Umum

1. Ikuti scope aturan: root, rak, dan buku.
2. Jangan memindahkan aturan antar scope tanpa diskusi.
3. Perubahan struktur global harus didokumentasikan di root docs.
4. Perubahan isi rak harus mengikuti shelves/<rak>/docs.
5. Perubahan isi buku harus mengikuti shelves/<rak>/<book>/docs.
