# Dokumentasi Project

Folder ini digunakan untuk menyimpan seluruh dokumentasi teknis dan manajemen project PostgreSQL Knowledge Base Rebuild.

### Cakupan Dokumentasi:
- **Workflow**: Prosedur kerja dan siklus pengembangan.
- **Roles**: Pembagian peran dan tanggung jawab dalam project.
- **Roadmap**: Rencana pengembangan jangka panjang dan jangka pendek.
- **Current Status**: Status terkini dari progres pengembangan.
- **Scope Guard**: Batasan-batasan kerja untuk menjaga fokus project.
- **Catatan Keputusan**: Dokumentasi mengenai alasan di balik keputusan teknis atau manajerial tertentu.
