# Naming Conventions (Global)

## Aturan Nama Folder

- Rak: `NN_topic-name` (contoh: `03_indexing`)
- Buku: `book-NN-topic-name` (contoh: `book-01-btree-index`)
- Dokumen aturan: gunakan `kebab-case.md`

## Aturan Nama File Utama

- `README.md` untuk pengantar scope
- `CHANGELOG.md` untuk riwayat perubahan (khusus buku)
- `docs/` untuk aturan dan penjelasan tambahan sesuai scope
