# Portable Project Instructions

Folder ini berisi kumpulan instruksi portabel (portable instructions) untuk PostgreSQL Living Knowledge Base. Berkas di dalam folder ini dirancang agar dapat disalin-tempel (copy-paste) secara langsung ketika pengguna membuat **ChatGPT Project** baru atau saat ingin menyetel ulang instruksi bawaan (Custom Instructions) di ruang obrolan (room) baru.

## Daftar Berkas Instruksi
1. [add-instructions-chatgpt-project.md](./add-instructions-chatgpt-project.md): Instruksi utama proyek untuk disalin ke kolom "Instructions" atau "Custom Instructions" pada ChatGPT Project.
2. [room-chat-00-instructions.md](./room-chat-00-instructions.md): Pedoman instruksi peran spesifik untuk **Room Chat 00** (Project Manager & Scope Guard).
3. [room-chat-01-analisa-saja-instructions.md](./room-chat-01-analisa-saja-instructions.md): Pedoman instruksi peran spesifik untuk **Room Chat 01** (Read-Only Analyst).
4. [gemini-3-flash-executor-instructions.md](./gemini-3-flash-executor-instructions.md): Pedoman instruksi peran spesifik untuk **Gemini 3 Flash** (Pure Executor).

## Cara Penggunaan
* Saat memulai ruang proyek baru di ChatGPT, unggah seluruh berkas di folder ini sebagai berkas referensi/pengetahuan (Knowledge files).
* Salin isi dari [add-instructions-chatgpt-project.md](./add-instructions-chatgpt-project.md) ke kolom Custom Instructions proyek Anda.
