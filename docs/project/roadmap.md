# Project Roadmap

Rencana pengembangan PostgreSQL Knowledge Base dalam 10 batch awal:

- **Batch 0**: Reset repository dan pembuatan struktur awal (Root, Materi, Docs).
- **Batch 1**: Pembuatan kerangka folder awal (Inisialisasi Materi).
- **Batch 2**: Kunci struktur Expert-Scale `materi/` (Rak 00-16).
- **Batch 3**: Refinement Rak 01 (Orientasi & Sejarah).
- **Batch 4 (Current)**: Kunci standar penulisan materi dan template.
- **Batch 5**: Sinkronisasi awal skeleton prioritas Level 0-4 berdasarkan standar.
- **Batch 6 - 9**: Mulai pengisian materi awal Level 0-4 secara bertahap.
- **Batch 10**: Sinkronisasi dokumentasi project (sync) dan evaluasi fase pertama.

*Catatan: Batch 5 tetap fokus pada struktur materi, bukan dokumentasi khusus.*
