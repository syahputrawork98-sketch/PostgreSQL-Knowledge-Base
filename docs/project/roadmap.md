# Project Roadmap

Rencana pengembangan PostgreSQL Knowledge Base dalam 10 batch awal:

- **Batch 0 (Current)**: Reset repository dan pembuatan struktur awal (Root, Materi, Docs).
- **Batch 1 - 5**: Fokus pada pembangunan kerangka folder dan sub-topik di dalam rak `materi/`.
- **Batch 6 - 9**: Mulai pengisian konten materi PostgreSQL awal secara bertahap.
- **Batch 10**: Sinkronisasi dokumentasi project secara menyeluruh dan evaluasi fase pertama.

*Catatan: Batch 5 tetap fokus pada struktur materi, bukan dokumentasi khusus.*
