# Scope Guard (Batasan Proyek)

Untuk menjaga fokus dan mencegah scope creep, batasan berikut harus dipatuhi:

- **Fokus Utama**: Saat ini hanya berfokus pada pengembangan `materi/` PostgreSQL.
- **Client/Server HOLD**: Dilarang mengerjakan aplikasi frontend (React/Vite) atau backend (Node/Express) sebelum materi siap.
- **Fitur Web HOLD**: Tidak membuat fitur web secara teknis (search, tagging, auth, dll) saat ini.
- **No Detailed Material (Phase-based)**: Tidak menulis materi panjang sebelum masuk ke batch pengisian materi yang ditentukan.
- **Struktur Tetap**: Tidak mengubah struktur besar (Rak 00-16) tanpa keputusan formal dari Room Chat 00/User.
- **No Extra Tech**: Tidak menambah teknologi lain seperti MongoDB atau Redis kecuali ditentukan di masa depan.
- **Source of Truth**: GitHub adalah sumber kebenaran tunggal setelah proses commit/push oleh user.
