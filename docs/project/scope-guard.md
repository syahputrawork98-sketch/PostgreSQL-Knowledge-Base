# Scope Guard (Batasan Proyek)

Untuk menjaga fokus dan mencegah scope creep, batasan berikut harus dipatuhi:

- **Dilarang** mengerjakan aplikasi client/frontend sebelum materi siap.
- **Dilarang** mengerjakan server/backend/API pada fase awal.
- **Dilarang** membuat fitur web (search, tagging, dll) secara teknis/kode saat ini.
- **Dilarang** masuk ke teknologi database lain (seperti MongoDB) kecuali ditentukan nanti.
- **Dilarang** mengisi materi PostgreSQL secara detail atau panjang lebar di Batch 0.
- **Fokus Utama**: Reset repository, pembuatan struktur awal, pembangunan kerangka materi, dan inisialisasi dokumentasi manajemen project.
