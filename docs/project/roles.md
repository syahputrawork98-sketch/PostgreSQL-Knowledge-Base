# Peran & Tanggung Jawab

Struktur tim dan peran dalam project ini:

- **User / Syah**: Owner project dan pengambil keputusan final. Menentukan visi dan arah utama.

- **Room Chat 01**: Analis read-only yang bertugas memberikan masukan dan analisis tambahan tanpa melakukan eksekusi langsung.
- **Gemini 3 Flash (Antigravity)**: Executor utama saat ini. Bertugas melakukan perubahan file, pembuatan struktur, dan penulisan konten berdasarkan instruksi.
- **Codex / VS Code**: Saat ini tidak digunakan secara aktif sebagai executor. Merupakan opsi masa depan jika diaktifkan oleh owner.
