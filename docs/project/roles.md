# Peran & Tanggung Jawab

Struktur tim dan peran dalam project ini:

- **User / Syah**: Owner project dan pengambil keputusan final. Menentukan visi dan arah utama.
- **Room Chat 00**: Manajer project, scope guard, pengelola roadmap, evaluator, dan pembuat instruksi final untuk executor.
- **Room Chat 01**: Analis/Konsultan diskusi yang bertugas memberikan masukan tambahan (Read-only/Consultative).
- **Gemini 3 Flash (Antigravity)**: Executor utama aktif saat ini. Bertugas melakukan perubahan file, pembuatan struktur, dan penulisan konten berdasarkan instruksi.
- **Codex / VS Code**: Tidak aktif saat ini sebagai executor.
