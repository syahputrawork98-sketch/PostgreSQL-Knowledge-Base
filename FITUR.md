# Rencana Fitur

Dokumen ini mencantumkan rencana fitur untuk PostgreSQL Knowledge Base. Saat ini, semua fitur berada dalam status **HOLD** atau **PLANNED** karena fokus utama adalah pembangunan struktur materi.

| Fitur | Deskripsi | Status |
|---|---|---|
| **Web Library Browser** | Antarmuka web untuk membaca materi dengan nyaman. | PLANNED / HOLD |
| **Rak dan Buku Materi** | Organisasi materi berbasis kategori (rak) dan topik (buku). | PLANNED / HOLD |
| **Search Materi** | Pencarian cepat di seluruh konten knowledge base. | PLANNED / HOLD |
| **Tag Materi** | Sistem pelabelan untuk mempermudah pengelompokan topik. | PLANNED / HOLD |
| **Bookmark/Checklist Belajar** | Melacak progres belajar pengguna. | PLANNED / HOLD |
| **Interview Preparation Mode** | Mode khusus untuk simulasi pertanyaan interview. | PLANNED / HOLD |
| **Version Update Tracker** | Notifikasi dan ringkasan fitur baru PostgreSQL. | PLANNED / HOLD |
| **Client App** | Aplikasi frontend (Vite/React). | PLANNED / HOLD |
| **Server/API** | Backend service untuk mengelola data dan konten. | PLANNED / HOLD |

---
*Catatan: Implementasi fitur akan dimulai setelah Batch 10.*
